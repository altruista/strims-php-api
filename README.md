## Nieoficjalne API dla Strims.pl przez cURL v1.04

#### API pozwala na:
- pobieranie wpisów
- dodawanie treści (link)
- dodawanie treści (tekst)
- dodawanie powiązanych linków
- dodawanie wpisów
- głosowanie (wpisy / treści)
- zamianę krótkich linków na pelne
